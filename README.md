# Database Penjualan Baju

Repository ini berisi implementasi basis data penjualan baju
online dan offline menggunakan MySQL.

## Fitur
- ERD & Normalisasi
- Foreign Key Constraint
- Trigger & Stored Procedure
- Data Dummy
- Query Laporan

## Cara Menjalankan
1. Jalankan file SQL secara berurutan dari folder `sql/`
- Databse & Create Table
- Data Dummy
- Database Programming (Advance)
- Manipulasi Data (CRUD & DML)
- Optimasi Query
2. Gunakan MySQL / MariaDB
3. jika ingin menghapus isi data file gunakan file `hapus isi table`
